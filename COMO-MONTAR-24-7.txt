2. 🌐 SUBIR A GITHUB

   - Copia los comandos que aparecen
   - Pégalos en PowerShell

3. 🚂 DEPLOYAR EN RAILWAY
   - Ve a: https://railway.app
   - Login con GitHub
   - "Deploy from GitHub repo"
   - Elige tu repo
   - Agrega PostgreSQL
   - Deploy!

4. 🎯 CONFIGURAR
   - Agrega variables de entorno
   - Ejecuta: npm run setup-db
   - ¡Listo!

Tiempo total: 15-20 minutos
